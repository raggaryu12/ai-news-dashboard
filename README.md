# AIニュースダッシュボード 🎣

夏・海・釣りをテーマにしたAIニュース収集ダッシュボード

## 機能

- 🔍 Google NewsのRSSフィードからAI関連ニュースを取得
- 🎨 夏・海・釣りをテーマにした爽やかなデザイン
- 🇯🇵 完全日本語対応のUI
- 🖱️ クリック可能なニュースカード
- ✨ ホバー時のアニメーション効果

## セットアップ

```bash
# 依存関係のインストール
pip install -r requirements.txt

# アプリの起動
streamlit run app.py
```

## 使い方

1. サイドバーの検索ボックスにキーワードを入力
2. ニュースが自動的に更新されます
3. カードをクリックして記事を読む

## 技術スタック

- Python
- Streamlit
- feedparser
